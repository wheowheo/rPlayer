# rPlayer 기능 검증 플랜

## 자동 테스트 (`scripts/test.sh`)

| # | 검증 항목 | 판정 기준 | 방법 |
|---|---------|----------|------|
| 1 | 릴리스 빌드 | 에러 0, 경고 0 | `cargo build --release` |
| 2 | 샘플 파일 존재 | 4개 + 4K 1개 | 파일 존재 확인 |
| 3 | H.264 720p 재생 | 파일 열기 + 렌더 + 오디오 | 로그 확인 |
| 4 | H.265 1080p 재생 | 파일 열기 + 렌더 + 오디오 | 로그 확인 |
| 5 | VP9 WebM 재생 | 파일 열기 + 렌더 + 오디오 | 로그 확인 |
| 6 | VFR 재생 | 파일 열기 + 렌더 + 오디오 | 로그 확인 |
| 7 | HW 디코딩 초기화 | "Hardware decoder" 로그 | VideoToolbox |
| 8 | 4K 60fps 성능 | render >= 55fps | fps 로그 |
| 9 | 한글 폰트 | "Loaded Korean font" | 폰트 경로 확인 |
| 10 | 재생 완료 감지 | "Playback finished" | 10초 파일 EOF |
| 11 | 오디오 DSP | "Audio output started" | 파이프라인 초기화 |
| 12 | GPU 어댑터 | "GPU adapter" | 디바이스 이름 |
| 13 | 패키징 | tar.gz + .app | `scripts/package.sh` |
| 14 | 배포 바이너리 | dist/ 바이너리로 재생 | 직접 실행 |

## 수동 테스트
